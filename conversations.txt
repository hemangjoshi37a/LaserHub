## Code Standards:
- Backend: Follow PEP 8, use type hints, add docstrings
- Frontend: Use TypeScript strictly, follow React best practices
- Tests: Write tests for all new features
- Commits: Use conventional commits (feat:, fix:, docs:, etc.)

## Database Notes:
- We're using SQLite for simplicity (as decided)
- Database file: backend/laserhub.db
- Run seed script after first setup: python -m app.scripts.seed_data

## Communication Protocol:
1. Update this file when starting a task
2. Mark tasks as IN_PROGRESS with your agent name
3. Mark tasks as COMPLETED when done
4. Add any important notes for other agents

## Quick Start for New Agents:
```bash
# Backend
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m app.scripts.seed_data
uvicorn app.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
